# <a name="ediscovery-solutions-in-microsoft-365"></a>Microsoft 365 の電子情報開示ソリューション

電子情報開示 (eDiscovery) とは、訴訟で証拠として使用できる電子的情報を特定および提供するプロセスです。 Microsoft 365 の電子情報開示ツールを使用して、Exchange Online メールボックス、Microsoft 365 グループ、Microsoft Teams、SharePoint Online と OneDrive for Business サイト、Skype for Business の会話、および Yammer チームのコンテンツを検索できます。 コンテンツ検索ツールを使用して、同じ電子情報開示検索でメールボックスとサイトを検索できます。 また、コア電子情報開示ケースを使用して、メールボックスやサイトにあるコンテンツを特定、保持、エクスポートできます。 組織に Office 365 E5 または Microsoft 365 E5 サブスクリプション (または関連する E5 アドオン サブスクリプション) がある場合は、Microsoft 365 の Advanced eDiscovery ソリューションを使用して、保管担当者をさらに管理し、コンテンツを分析できます。
  
Microsoft 365 には、次の電子情報開示ツールが提供されています。
  
- [コンテンツ検索](#content-search)

- [コア電子情報開示](#core-ediscovery)

- [Advanced eDiscovery](#advanced-ediscovery)

## <a name="advanced-ediscovery"></a>Advanced eDiscovery

Microsoft 365 の Advanced eDiscovery ソリューション *(Advanced eDiscovery v2.0* とも呼ばれる) は、Microsoft 365 の既存の電子情報開示および分析機能を基に構築されています。 この電子情報開示ソリューションは、組織の内部および外部の調査に対応するコンテンツを保持、収集、レビュー、分析、エクスポートするためのエンドツーエンドのワークフローを提供します。 また、法務チームはカストディアンを管理し、法的情報保留通知ワークフロー全体を管理して、ケースに関与するカストディアンと通信できます。
