# <a name="overview-of-microsoft-365-business-premium-setup"></a>Microsoft 365 Business Premium セットアップの概要

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4jZwg?autoplay=false]

Microsoft 365 でビジネスをクラウドに移行する決定をおめでとうございます。 ビジネスに 1 人か 20 人か、少し計画を立て、Microsoft 365 Business Premium を有効に利用できます。

まず、次の&#39;取得する情報&#39;確認します。

- Office、Excel、PowerPoint、Outlook のようなアプリケーションを使用します。
- ユーザーと従業員の個人設定された電子メール アドレス (rob@contoso.com)。
- クラウド内のドキュメント ストレージ (OneDrive と Microsoft Teams) など。
- Windows 10 Pro へのアップグレードは、Windows 7 Professional、Windows 8、Windows 8.1 Pro から行います。
- エンタープライズ レベルのセキュリティを備え、すべての iOS、Android、および Windows 10 デバイス上の作業ファイルを保護します。

会計&#39;、CRM、またはパブリック Web サイト用のアプリを取得していない可能性があります。

> [!NOTE]

## <a name="1-sign-up-for-the-cloud"></a>1. クラウドにサインアップする

ビジネス向けクラウド サブスクリプションを選択する場合は、サブスクリプションで必要な機能について選択できます。 Microsoft の小規模 [ビジネス プランについて説明します](https://www.microsoft.com/microsoft-365/business?rtc=1)。

- メール、Office作成、コラボレーション、クラウド ストレージ用のオンライン アプリ (Web ブラウザーでアクセスできるアプリ) が必要な場合は [、[365 Business Essentials]](https://www.microsoft.com/en-us/p/office-365-business-essentials/cfq7ttc0k59v?rtc=1&amp;activetab=pivot:overviewtab) を選択します。 これには、Word Online、Excel Online、PowerPoint Online、および SharePoint Online が含まれます。
- Pc Office と Mac で直接実行する Word、Excel、Teams など、機能豊富なバージョンの Office アプリが必要な場合は [、[365 Business Premium]](https://products.office.com/en-us/business/office-365-business-premium) を選択します。 これは、オンライン アプリに加えてです。
- すべてのアプリに加えて、クラウド セキュリティとデバイス管理をOffice Microsoft [365](https://www.microsoft.com/microsoft-365/business?rtc=1) Business を選択します。 これには、ウイルス、マルウェア、ランサムウェアなどの保護、デバイスの管理と暗号化が含まれます。 また、以前の Windows Professional バージョンから Windows 10 Pro に無料でアップグレードできます。
- ビジネスが大きい場合や、より複雑な IT アプリやサービスが必要な場合は [、Microsoft 365 Enterprise](https://www.microsoft.com/microsoft-365/compare-all-microsoft-365-plans)プランについて説明します。


## <a name="2-add-a-domain-name"></a>2. ドメイン名を追加する

## <a name="3-add-users-with-licenses"></a>3. ライセンスを持つユーザーを追加する

- ビジネス ソフトウェアとサービスを使用する必要がある会社のすべての従業員または担当者に対して、Microsoft 365 でユーザーとして追加します。 自分に割り当てるのと同じライセンス (Microsoft 365 Business Premium など) を割り当てるか、複数または少ない機能 (Microsoft 365 Business Basic など) で異なるプランを割り当てできます。
- ユーザーを作成すると、ユーザー&#39;、仕事用メール アドレス、パスワード、その他の情報を追加できます。 この&#39;を使用してサービスにサインインし、電子メールを受信します。 パスワード、アクセス許可、その他の設定は、いつでも変更できます。


## <a name="4-enable-threat-protection"></a>4. 脅威保護を有効にする

- サービス&#39;高度な脅威保護を有効にする必要があります。 Microsoft 365 Advanced Threat Protection には、安全な添付ファイル、安全なリンク、SharePoint、OneDrive、Microsoft Teams の保護、フィッシング対策保護、リアルタイムの脅威検出などの構成、保護、検出機能が含まれます。
- 既定では、一般的&#39;の脅威に対するコア保護に関するヘルプを受け取りますが、これらの設定は、後でビジネス ニーズに合わせてカスタマイズできます。

## <a name="5-manage-and-secure-your-devices"></a>5. デバイスの管理とセキュリティ保護

- Microsoft 365 Business Premium を使用すると、デバイスの保護、オンラインの脅威からの保護、偶発的なデータ損失からの保護を受けられます。 また、Windows 10 Pro Windows Defenderアプリやアプリの自動更新、BitLocker、自動更新Officeもあります。
- 多要素認証 (MFA)、電子メール暗号化、マルウェア保護、ランサムウェア保護をセットアップして、デバイスやデータへの不正アクセスを防止できます。
- PC、SharePoint または OneDrive for Business、またはモバイル デバイスにローカルに保存されている貴重な作業データを保護できます。 Microsoft 365 用の組み込みのモバイル デバイス管理 (MDM) は、iPhone、iPad、Android、Windows 電話など、&#39; モバイル デバイスのユーザーのセキュリティ保護と管理に役立ちます。 MDM を使用すると、デバイスのセキュリティ ポリシーを作成および管理したり、リモートでデバイスをワイプしたり、詳細なデバイス レポートを参照できます。

## <a name="6-upgrade-your-pcs-to-windows-10-pro"></a>6. PC を Windows 10 Pro にアップグレードする

- 高度なセキュリティやデバイス管理など、すべての Microsoft 365 Business Premium 機能を利用できるよう、PC の Windows 10 Professional へのアップグレードを開始します (Windows 7 Pro、Windows 8 Pro、Windows 8.1 Pro を実行し続ける場合は、Microsoft 365 Business&#39;Premium を無料で使用できます)。
- また、コンシューマー バージョンから Windows 10 Pro にアップグレードするか、Microsoft 365 Business Standard&#39;を実行する必要がある場合でも、1 回の購入が必要です。

## <a name="7-connect-your-pcs-to-microsoft-365"></a>7. PC を Microsoft 365 に接続する

- Microsoft 365 を使い切&#39;、Windows 10 PC を Microsoft 365 Business Premium と Azure Active Directory に接続します。 これにより、お使いの PC を Microsoft 365 Business Premium によって管理し、ソフトウェア更新プログラムを取得し、データ損失や他のサイバー脅威に対する保護を受け取ります。
- PC を接続すると、Windows 10 はデバイスを組織&#39;のネットワークに登録し、デバイスをネットワークに参加し、組織&#39;のユーザー名とパスワードを使用してリソースにサインインしてアクセスできます。
- 組織の要件に&#39;、Microsoft Intune などの 2 段階認証、暗号化、モバイル デバイス管理の設定を求める場合があります。

## <a name="9-migrate-old-email-and-files"></a>9. 古いメールとファイルを移行する

- ある時点で、&#39;メール、予定表、連絡先、ビジネス ドキュメント、その他のファイルを Microsoft 365 Business Premium にインポートする必要があります。
- 電子メールの場合は、古いメールを PST としてエクスポートし、その PST を新しいメール アカウントにインポートすることで、Outlook でこれを行います。 その後、他のコンピューターでこれを繰り返します。 パートナーを使用して、この手順を支援できます。
