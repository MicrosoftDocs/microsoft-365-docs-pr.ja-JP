# <a name="cancel-google-workspace-subscription"></a>Google Workspace サブスクリプションをキャンセルする

1. Google Admin に [サインインし、[](https://admin.google.com/) 課金] に移動 **します**。
1. [サブスクリプション **] ボックスの一覧** で、G Suite サブスクリプション、[サブスクリプションのキャンセル] の順に選択し、[続行] を **選択します**。

    > [!IMPORTANT]
    > サブスクリプションはすぐに取り消され、この操作を元に戻すことはできません。

    :::image type="content" source="../../media/cancelgoogleworspace.png" alt-text="[Google Admin 課金] ページのスクリーン ショット。":::

DNS ホスト プロバイダーが Google ドメインではない場合、サブスクリプションの一覧にドメイン登録が表示されない場合があります。
