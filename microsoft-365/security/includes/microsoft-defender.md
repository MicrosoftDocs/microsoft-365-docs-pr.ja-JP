> [!IMPORTANT]
> 改善された [Microsoft 365 セキュリティ センター](https://security.microsoft.com)がパブリック プレビューで利用できるようになりました。 この新しいエクスペリエンスにより、Defender for Endpoint、Defender for Office 365、Microsoft 365 Defender などが Microsoft 365 セキュリティ センターに導入されます。 セキュリティ チームは、別の製品ポータルに移動することなく、すべてのエンドポイント、メール、製品間の調査、構成、修復を管理できるようになりました。 [変更点についての詳細情報](https://docs.microsoft.com/microsoft-365/security/mtp/overview-security-center) はこちらを参照してください。
