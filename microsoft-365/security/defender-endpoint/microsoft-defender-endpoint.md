<span data-ttu-id="e71fd-112">Defender for Endpoint では、Windows 10 に組み込むテクノロジと Microsoft の堅牢なクラウド サービスの組み合わせを使用します。</span><span class="sxs-lookup"><span data-stu-id="e71fd-112">Defender for Endpoint uses the following combination of technology built into Windows 10 and Microsoft's robust cloud service:</span></span>
