## <a name="microsoft-365-defender"></a>Microsoft 365 Defender

[Microsoft 365 Defender](./microsoft-threat-protection.md) は、侵害前および侵害後の統合エンタープライズ防御スイートです。 エンドポイント、ID、電子メール、およびアプリケーション間で検出、防止、調査、応答をネイティブに調整し、高度な攻撃に対する統合保護を提供します。

Microsoft 365 Defender は [、Microsoft Defender for](https://www.microsoft.com/microsoft-365/windows/microsoft-defender-atp)Endpoint、Microsoft [Defender for Office 365、Microsoft](https://www.microsoft.com/microsoft-365/exchange/advance-threat-protection) [Defender for Identity、Azure](https://azure.microsoft.com/features/azure-advanced-threat-protection/) [Active Directory Identity Protection、Microsoft Cloud](/azure/active-directory/identity-protection/overview-identity-protection)App [Security](https://www.microsoft.com/microsoft-365/enterprise-mobility-security/cloud-app-security) の機能を 1 つのソリューションに結合します。

### <a name="mitre-demonstrated-real-world-detection-response-and-protection-from-advanced-attacks"></a>MITRE: 高度な攻撃からの実際の検出、応答、および保護の実証

MITRE のテスト アプローチの中核となるのは、ソリューションが適切に検出して対応できるかどうかを理解するために、実際の攻撃を組み込む方法です。 このテストではエンドポイントの検出と応答に重点を置いた一方で、MITRE のシミュレートされた APT29 攻撃は複数の攻撃ドメインにまたがるので、エンドポイント保護以外の防御側に権限を与える機会が生み出されます。 Microsoft は、Microsoft 365 Defender を使用して、エンドポイントを超えて防御側の可視性を拡大しました。

- ATT&CK ベースの Microsoft 365 Defender の評価 - 2020 年 5 月: 実際の検出[をリード](https://www.microsoft.com/security/blog/2020/05/01/microsoft-threat-protection-leads-real-world-detection-mitre-attck-evaluation/)する

    Microsoft 365 Defender は、攻撃チェーンのステージ全体で約 100% の範囲を提供しました。 これは、攻撃者のアクティビティに対する一流の可視性を提供しました。 可視性により、特定の構成変更に依存したセキュリティ 運用センターとベンダー ソリューションの手動作業が大幅に削減されます。 また、Microsoft 365 Defender では可視性のギャップが最も少なく、検出されない操作を行う攻撃者の能力が低下しました。

## <a name="endpoint-detection--response"></a>エンドポイント検出&応答

Microsoft Defender for Endpoint endpoint detection and response [capabilitis](/windows/security/threat-protection/microsoft-defender-atp/overview-endpoint-detection-response) provide advanced attack detections are near real-time and actionable. セキュリティ アナリストは、効率的にアラートの優先順位を設定し、違反の全容を可視化して、脅威に対処する対応策を講じることができます。

Microsoft Defender for Endpoint の EDR およびエンドポイント保護機能は、業界のテストと出版物から肯定的な結果を受け取っています。 [SC Labs は](https://www.scmagazine.com/home/reviews/sc-product-reviews-endpoint-security/) 2020 年 6 月にエンドポイント セキュリティ ツールを評価し、Microsoft Defender for Endpoint [5/5 の星を獲得しました](https://www.scmagazine.com/review/microsoft-defender-advanced-threat-protection/)。 Microsoft Defender for Endpoint は、セキュリティ機能の完全なセットを使用して、組織を最新の脅威の状況から保護する機能を呼び出しました。 また、SC Labs はエンドポイント セキュリティ ソリューションを総合的かつ統合的に識別しました。 また、攻撃チェーンがソリューションによって完全にカバーされる中、エンドポイントの検出および応答機能によるエンドポイント保護の収束も確認しました。

### <a name="mitre-industry-leading-optics-and-detection-capabilities"></a>MITRE: 業界をリードする光学および検出機能

MITRE は、標的型攻撃グループ APT3 (ボロンまたは UPS とも呼ばれる) で一般的に使用される手法を検出する製品の機能をテストしました。 検出機能を分離するために、すべての保護機能と防止機能がオフにされています。 Microsoft は、ATT および CK フレームワークに基づく MITRE 評価にサインアップする最初の EDR ベンダーの 1&です。 このフレームワークは、今日、攻撃者の手法と戦術の最も包括的なカタログと広く見なされています。

- ATT&エンドポイント向け Microsoft Defender の CK ベースの評価 - 2018 年 12 月: 主要な光学機能と検出[機能](https://www.microsoft.com/security/blog/2018/12/03/insights-from-the-mitre-attack-based-evaluation-of-windows-defender-atp/)  |  [分析](https://techcommunity.microsoft.com/t5/Windows-Defender-ATP/MITRE-evaluation-highlights-industry-leading-EDR-capabilities-in/ba-p/369831)

    Microsoft Defender for Endpoint は、攻撃チェーン全体にわたって攻撃者の手法に関する包括的な範囲を提供しました。 注目されたのは、テレメトリの幅、脅威インテリジェンスの強さ、機械学習、ヒューリスティック、および動作監視による高度な自動検出です。

## <a name="to-what-extent-are-tests-representative-of-protection-in-the-real-world"></a>実際の世界で保護を代表するテストは、どの程度ですか?

独立したセキュリティ業界のテストでは、最適なウイルス対策製品とセキュリティ製品を不偏な方法で評価します。 ただし、Microsoft では、この記事で強調表示されている評価でテストされる範囲を超えて、より広範で広範な脅威が見受け取ります。 平均月に、Microsoft のセキュリティ製品は 1 億を超える新しい脅威を特定します。 独立したテスターがこれらの脅威の 1% を取得してテストできる場合でも、20 製品または 30 製品で 100 万回のテストです。 言い換えれば、マルウェアの環境の広大さは、現実世界の脅威に対する保護の品質を評価するのが難しくなります。

Microsoft Defender for Endpoint 内[](https://cloudblogs.microsoft.com/microsoftsecure/2017/12/11/detonating-a-bad-rabbit-windows-defender-antivirus-and-layered-machine-learning-defenses)の機能は、業界のウイルス対策テストに考慮されていない保護の層を追加し、最新の高度な脅威の一部に対処します。 Defender for Endpoint の残りの部分から AV を分離すると、Microsoft のセキュリティ スタックが実際の世界でどのように動作するのかの部分的な画像が作成されます。 たとえば、攻撃表面の縮小とエンドポイント&機能は、マルウェアが最初にデバイスにアクセスするのを防ぐのに役立ちます。 Microsoft Defender for Endpoint コンポーネントは [、Microsoft Defender Antivirus](https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RE2ouJA) がこれらの業界テストで見逃したサンプルをキャッチしているという実績があります。 これは、Microsoft のセキュリティ スイートが実際の世界の顧客をどれだけ効果的に保護するのかのより代表的な機能です。
