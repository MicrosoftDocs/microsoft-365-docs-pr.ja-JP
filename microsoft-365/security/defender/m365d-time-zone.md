# <a name="set-the-time-zone-for-microsoft-365-defender"></a>Defender のタイム ゾーンをMicrosoft 365する

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**適用対象:**
- Microsoft 365 Defender



Microsoft 365Defender は、ローカル タイム ゾーンまたは UTC を使用して日付と時刻の情報を表示できます。 選択したタイムゾーンは、Microsoft 365 セキュリティ センターの次の機能に示される日付と時刻の情報に適用されます。
- インシデント
- アクション センターを含む調査と修復の自動化
- 高度な捜索

これらの機能のタイム ゾーンを設定するには、[Defender **タイム** ゾーン設定Microsoft 365  >  **に**  >  **移動します**。
