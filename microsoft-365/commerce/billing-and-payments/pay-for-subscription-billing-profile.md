# <a name="how-to-pay-for-your-subscription-with-a-billing-profile"></a>課金プロファイルを使用してサブスクリプションの支払いを行う方法

::: moniker range="o365-21vianet"

> [!NOTE]
> 管理センターは変更中です。 エクスペリエンスがここで説明されている詳細と一致しない場合は、「[新しい Microsoft 365 管理センターについて](https://docs.microsoft.com/microsoft-365/admin/microsoft-365-admin-center-preview?view=o365-21vianet&preserve-view=true)」を参照してください。

::: moniker-end

サブスクリプションを購入する場合は、課金プロファイルで支払います。 請求プロファイルは特定の支払い方法にリンクされ、クレジット カード、デビット カード、または請求書の場合がありますが、銀行口座にはリンクされません。

アカウントに請求プロファイルが設定されているのが不明な場合は、「請求プロファイル [について」を参照してください](manage-billing-profiles.md)。 請求プロファイルをお持ちない場合は、サブスクリプションの支払 [い方法をご覧ください](pay-for-your-subscription.md)。

## <a name="paying-with-recurring-billing-turned-on-or-off"></a>継続請求を有効または無効にした支払い

既定では、継続請求を使用しているすべての有料サブスクリプションで、継続請求が自動的に有効になります。 請求期間ごとに、その請求プロファイルを使用するサブスクリプションに対して支払いを行う請求プロファイルに関連付けられている支払い方法が自動的に請求されます。 支払い方法が拒否された場合は、請求書の[今すぐ支払い] ボタンを使用して、サブスクリプションの 1 回払いを行います。

請求プロファイルで継続請求がオフになっている場合は、請求プロファイルにリンクされている支払い方法に関係なく、請求書の [今すぐ支払う] ボタンを使用して、請求期間ごとに支払います。 支払いは、チェックまたは電子資金移動 (EFT) でも行います。 手順については、請求書の PDF コピーに記載されています。

## <a name="paying-by-invoice"></a>請求書による支払い

請求書で支払う設定の請求プロファイルがある場合は、チェックまたは EFT を使ってサブスクリプションの支払いを行います。 請求書の [今すぐ支払う] ボタンを使用して、クレジット カードを使用してオンライン支払いを行う方法も可能です。

請求書による支払いを受け取る資格を得るには、次の条件を満たしている必要があります。

- 確立された顧客になる
- サブスクリプションのコストが一定の金額を超えている (この金額はサービスの場所によって異なります)
- 与信確認に合格する

与信確認が必要な場合は、サブスクリプションを購入すると通知されます。 連絡を受け取る場合は、与信承認の申請に関する詳細な情報が記載されたメールが届きます。 与信確認は通常、2 営業日以内に完了します。

請求プロファイルが請求書に裏付けられている場合は、請求明細書を表示する準備が整ったときにメールが届きます。 このメールには、請求明細書のコピーが含まれている必要があります。 ただし、請求明細書の [コピーをメールで受け取る方法を選択できます](view-your-bill-or-invoice.md#receive-a-copy-of-your-billing-statement-in-email)。 請求明細書には、支払いを行うオプションと支払い方法に関する詳細が含まれています。 請求プロファイルに発注書 (PO) 番号を入力すると、その番号が請求明細書に表示されます。 請求明細書へのアクセスの詳細については、「[課金内容または請求書を表示する](view-your-bill-or-invoice.md)」を参照してください。

## <a name="can-i-pay-my-invoice-online"></a>請求書をオンラインで支払できますか?

請求プロファイルで継続請求がオフになっている場合は、クレジット カードを使用して請求書をオンラインで支払えます。 支払いを行う場合は、Microsoft 365 管理センターで請求書の [今すぐ支払う] ボタンを使用します。 請求書を見つけるには、「請求書または請求書 [を表示する」を参照してください](view-your-bill-or-invoice.md)。

## <a name="can-i-change-from-my-current-payment-method-to-paying-by-invoice"></a>現在の支払い方法から請求書での支払い方法に変更できますか?

請求プロファイルがクレジット カードまたはデビット カードに裏付けられている場合は、支払い方法を別のクレジット カードまたはデビット カードにのみ変更できます。 請求書での支払いには変更できない。

## <a name="can-i-change-from-paying-by-invoice-to-using-a-different-payment-method"></a>請求書支払いから別の支払い方法への変更はできますか?

請求プロファイルが請求書支払いによって裏付けられている場合は、支払い方法を変更できます。 請求書の [今すぐ支払 **う** ] ボタンを使用して、クレジット カードまたはデビット カードで、またはチェック または EFT で支払います。
