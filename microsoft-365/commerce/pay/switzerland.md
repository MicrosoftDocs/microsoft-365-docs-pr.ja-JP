Bank: Citibank, London\
SWIFT コード: CITIGB2L\
IBAN コード: GB44CITI18500812207648\
アカウント番号: 12207648\
アカウント名: MICROSOFT

Bank: Citibank Europe plc\
SWIFT コード: CITIIE2X\
IBAN コード: IE59CITI99005170000132\
アカウント番号: 70000132\

現地の Microsoft 子会社から発行された請求書は、VAT/売上税/GST の標準税率で発行され、VAT/売上税/GST の目的で国内のサービス供給と見なされます。
