# <a name="problems-with-your-microsoft-365-for-business-product-key"></a>ビジネス向け Microsoft 365 プロダクト キーに関する問題

一般ビジネス向け **Microsoft 365** のプロダクト キーを使用しようとするときにエラー メッセージが表示される場合は、次の一般的な解決策を試してください。 
  
 **すぐにヘルプが必要な場合** [Microsoft サポートに問い合わせください](../admin/contact-support-for-business-products.md)。 
  
 **Microsoft 365 ファミリまたは Microsoft 365 Personal** プロダクト キーのヘルプについては、「Microsoft 365 ファミリまたは Microsoft 365 個人用プロダクト キーでプロダクト キーを使用する [Office。](https://support.microsoft.com/office/12a5763a-d45c-4685-8c95-a44500213759.aspx)
