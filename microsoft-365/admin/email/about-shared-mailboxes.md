# <a name="about-shared-mailboxes"></a><span data-ttu-id="f2d67-104">共有メールボックスについて</span><span class="sxs-lookup"><span data-stu-id="f2d67-104">About shared mailboxes</span></span>

<span data-ttu-id="f2d67-105">共有メールボックスは、会社情報やサポート メール アドレス、受付デスク、または複数のユーザーが共有する可能性のあるその他の機能など、複数のユーザーが同じメールボックスにアクセスする必要がある場合に使用されます。</span><span class="sxs-lookup"><span data-stu-id="f2d67-105">Shared mailboxes are used when multiple people need access to the same mailbox, such as a company information or support email address, reception desk, or other function that might be shared by multiple people.</span></span>

<span data-ttu-id="f2d67-106">グループ メールボックスへのアクセス許可を持つユーザーは、管理者がそのユーザーに対して必要なアクセス許可を付与している場合、メールボックスのメール アドレスとして送信したり、メールボックスのメール アドレスの代わりに送信したりできます。</span><span class="sxs-lookup"><span data-stu-id="f2d67-106">Users with permissions to the group mailbox can send as or send on behalf of the mailbox email address if the administrator has given that user permissions to do that.</span></span> <span data-ttu-id="f2d67-107">これは、ユーザーが "Contoso サポート" や "建物 A の受付デスク" からメールを送信できるため、ヘルプとサポートのメールボックスとして特に役立ちます。</span><span class="sxs-lookup"><span data-stu-id="f2d67-107">This is particularly useful for help and support mailboxes because users can send emails from "Contoso Support" or "Building A Reception Desk."</span></span>

## <a name="before-you-begin"></a><span data-ttu-id="f2d67-108">はじめに</span><span class="sxs-lookup"><span data-stu-id="f2d67-108">Before you begin</span></span>

- <span data-ttu-id="f2d67-110">**ライセンス:** 共有メールボックスには、ライセンスを割り当てなくても最大 50 GB のデータを保存できます。</span><span class="sxs-lookup"><span data-stu-id="f2d67-110">**Licenses:** Your shared mailbox can store up to 50GB of data without you assigning a license to it.</span></span> <span data-ttu-id="f2d67-111">その後に、より多くのデータを格納するには、メールボックスにライセンスを割り当てる必要があります。</span><span class="sxs-lookup"><span data-stu-id="f2d67-111">After that, you need to assign a license to the mailbox to store more data.</span></span> <span data-ttu-id="f2d67-112">共有メールボックス ライセンスの詳細については、「[Exchange Online の制限](/office365/servicedescriptions/exchange-online-service-description/exchange-online-limits#StorageLimits)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="f2d67-112">For more details on shared mailbox licensing, please see [Exchange Online Limits](/office365/servicedescriptions/exchange-online-service-description/exchange-online-limits#StorageLimits).</span></span> <span data-ttu-id="f2d67-113">共有メールボックスが記憶域の上限に達した場合、しばらくはメールを受信できますが、新しいメールを送信できなくなります。</span><span class="sxs-lookup"><span data-stu-id="f2d67-113">When a shared mailbox reaches the storage limit, you'll be able to receive email for a while, but you won't be able to send new email.</span></span> <span data-ttu-id="f2d67-114">その後、メールも受信できなくなります。</span><span class="sxs-lookup"><span data-stu-id="f2d67-114">Then, after that, it will stop receiving email.</span></span> <span data-ttu-id="f2d67-115">そのメールボックスにメールを送信したユーザーには、配信不能エラーが送信されます。</span><span class="sxs-lookup"><span data-stu-id="f2d67-115">Senders to the mailbox will get a non-delivery receipt.</span></span>

- <span data-ttu-id="f2d67-116">**ユーザーのアクセス許可:** 共有メールボックスを使用するには、ユーザーにアクセス許可 (メンバーシップ) を付与する必要があります。</span><span class="sxs-lookup"><span data-stu-id="f2d67-116">**User permissions:** You need to give users permissions (membership) to use the shared mailbox.</span></span> <span data-ttu-id="f2d67-117">組織内のユーザーのみが共有メールボックスを使用できます。</span><span class="sxs-lookup"><span data-stu-id="f2d67-117">Only people inside your organization can use a shared mailbox.</span></span>

- <span data-ttu-id="f2d67-118">**外部ユーザー:** 組織外のユーザー (Gmail アカウントを持つユーザーなど) に共有メールボックスへのアクセスを許可することはできません。</span><span class="sxs-lookup"><span data-stu-id="f2d67-118">**External users:** You can't give people outside your business (such as people with a Gmail account) access to your shared mailbox.</span></span> <span data-ttu-id="f2d67-119">この操作を行う必要がある場合は、代わりに Outlook 用グループを作成することを検討してください。</span><span class="sxs-lookup"><span data-stu-id="f2d67-119">If you want to do this, consider creating a group for Outlook instead.</span></span> <span data-ttu-id="f2d67-120">詳細については、「[管理センターで Microsoft 365 グループを作成する](../create-groups/create-groups.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="f2d67-120">To learn more, see [Create a Microsoft 365 group in the admin center](../create-groups/create-groups.md).</span></span>

- <span data-ttu-id="f2d67-125">**暗号化:** 共有メールボックスから送信されたメールを暗号化することはできません。</span><span class="sxs-lookup"><span data-stu-id="f2d67-125">**Encryption:** You can't encrypt email sent from a shared mailbox.</span></span> <span data-ttu-id="f2d67-126">共有メールボックスには独自のセキュリティ コンテキスト (ユーザー名とパスワード) がないため、キーを割り当てることができないためです。</span><span class="sxs-lookup"><span data-stu-id="f2d67-126">This is because a shared mailbox does not have its own security context (username/password) so it cannot be assigned a key.</span></span> <span data-ttu-id="f2d67-127">2 人以上のユーザーがメンバーで、メンバーが独自のキーを使用して暗号化してメールを送受信した場合、他のメンバーはメールを読むことができる場合もあれば、できない場合もあります。メールがどのパブリック キーで暗号化されたかに依存します。</span><span class="sxs-lookup"><span data-stu-id="f2d67-127">If more than one person is a member, and they send/receive emails they encrypted with their own keys, other members might be able to read the email and others might not, depending which public key the email was encrypted with.</span></span>

- <span data-ttu-id="f2d67-128">**メールボックスの変換:** ユーザー メールボックスを、共有メールボックスに変換することができます。</span><span class="sxs-lookup"><span data-stu-id="f2d67-128">**Mailbox conversion:** You can convert user mailboxes to shared mailboxes.</span></span> <span data-ttu-id="f2d67-129">「[ユーザー メールボックスを共有メールボックスに変換する](convert-user-mailbox-to-shared-mailbox.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="f2d67-129">See [Convert a user mailbox to a shared mailbox](convert-user-mailbox-to-shared-mailbox.md).</span></span>
