# <a name="about-shared-mailboxes"></a><span data-ttu-id="4d694-104">共有メールボックスについて</span><span class="sxs-lookup"><span data-stu-id="4d694-104">About shared mailboxes</span></span>

<span data-ttu-id="4d694-105">共有メールボックスは、会社情報やサポート メール アドレス、受付デスク、または複数のユーザーが共有する可能性のあるその他の機能など、複数のユーザーが同じメールボックスにアクセスする必要がある場合に使用されます。</span><span class="sxs-lookup"><span data-stu-id="4d694-105">Shared mailboxes are used when multiple people need access to the same mailbox, such as a company information or support email address, reception desk, or other function that might be shared by multiple people.</span></span>

<span data-ttu-id="4d694-106">グループ メールボックスへのアクセス許可を持つユーザーは、管理者がそのユーザーに対して必要なアクセス許可を付与している場合、メールボックスのメール アドレスとして送信したり、メールボックスのメール アドレスの代わりに送信したりできます。</span><span class="sxs-lookup"><span data-stu-id="4d694-106">Users with permissions to the group mailbox can send as or send on behalf of the mailbox email address if the administrator has given that user permissions to do that.</span></span> <span data-ttu-id="4d694-107">これは、ユーザーが "Contoso サポート" や "建物 A の受付デスク" からメールを送信できるため、ヘルプとサポートのメールボックスとして特に役立ちます。</span><span class="sxs-lookup"><span data-stu-id="4d694-107">This is particularly useful for help and support mailboxes because users can send emails from "Contoso Support" or "Building A Reception Desk."</span></span>

- <span data-ttu-id="4d694-109">**ライセンス:** 共有メールボックスには、ライセンスを割り当てなくても最大 50 GB のデータを保存できます。</span><span class="sxs-lookup"><span data-stu-id="4d694-109">**Licenses:** Your shared mailbox can store up to 50GB of data without you assigning a license to it.</span></span> <span data-ttu-id="4d694-110">その後に、より多くのデータを格納するには、メールボックスにライセンスを割り当てる必要があります。</span><span class="sxs-lookup"><span data-stu-id="4d694-110">After that, you need to assign a license to the mailbox to store more data.</span></span> <span data-ttu-id="4d694-111">共有メールボックス ライセンスの詳細については、「[Exchange Online の制限](/office365/servicedescriptions/exchange-online-service-description/exchange-online-limits#StorageLimits)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="4d694-111">For more details on shared mailbox licensing, please see [Exchange Online Limits](/office365/servicedescriptions/exchange-online-service-description/exchange-online-limits#StorageLimits).</span></span> <span data-ttu-id="4d694-112">共有メールボックスが記憶域の上限に達した場合、しばらくはメールを受信できますが、新しいメールを送信できなくなります。</span><span class="sxs-lookup"><span data-stu-id="4d694-112">When a shared mailbox reaches the storage limit, you'll be able to receive email for a while, but you won't be able to send new email.</span></span> <span data-ttu-id="4d694-113">その後、メールも受信できなくなります。</span><span class="sxs-lookup"><span data-stu-id="4d694-113">Then, after that, it will stop receiving email.</span></span> <span data-ttu-id="4d694-114">そのメールボックスにメールを送信したユーザーには、配信不能エラーが送信されます。</span><span class="sxs-lookup"><span data-stu-id="4d694-114">Senders to the mailbox will get a non-delivery receipt.</span></span>

- <span data-ttu-id="4d694-115">**ユーザーのアクセス許可:** 共有メールボックスを使用するには、ユーザーにアクセス許可 (メンバーシップ) を付与する必要があります。</span><span class="sxs-lookup"><span data-stu-id="4d694-115">**User permissions:** You need to give users permissions (membership) to use the shared mailbox.</span></span> <span data-ttu-id="4d694-116">組織内のユーザーのみが共有メールボックスを使用できます。</span><span class="sxs-lookup"><span data-stu-id="4d694-116">Only people inside your organization can use a shared mailbox.</span></span>

- <span data-ttu-id="4d694-117">**外部ユーザー:** 組織外のユーザー (Gmail アカウントを持つユーザーなど) に共有メールボックスへのアクセスを許可することはできません。</span><span class="sxs-lookup"><span data-stu-id="4d694-117">**External users:** You can't give people outside your business (such as people with a Gmail account) access to your shared mailbox.</span></span> <span data-ttu-id="4d694-118">この操作を行う必要がある場合は、代わりに Outlook 用グループを作成することを検討してください。</span><span class="sxs-lookup"><span data-stu-id="4d694-118">If you want to do this, consider creating a group for Outlook instead.</span></span> <span data-ttu-id="4d694-119">詳細については、「[管理センターで Microsoft 365 グループを作成する](../create-groups/create-groups.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="4d694-119">To learn more, see [Create a Microsoft 365 group in the admin center](../create-groups/create-groups.md).</span></span>

- <span data-ttu-id="4d694-120">**Outlook での使用:** ブラウザーから Outlook on the web を使用して共有メールボックスにアクセスするだけでなく、Outlook for iOS アプリまたは Outlook for Android アプリを使用することもできます。</span><span class="sxs-lookup"><span data-stu-id="4d694-120">**Use with Outlook:** In addition to using Outlook on the web from your browser to access shared mailboxes, you can also use the Outlook for iOS app or the Outlook for Android app.</span></span> <span data-ttu-id="4d694-121">詳細については、「[Outlook Mobile に共有メールボックスを追加する](https://support.microsoft.com/office/f866242c-81b2-472e-8776-6c49c5473c9f)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="4d694-121">To learn more, see [Add a shared mailbox to Outlook mobile](https://support.microsoft.com/office/f866242c-81b2-472e-8776-6c49c5473c9f).</span></span> <span data-ttu-id="4d694-122">もう 1 つのオプションは、共有メールボックスのグループを作成することです。</span><span class="sxs-lookup"><span data-stu-id="4d694-122">Another option is to create a group for your shared mailbox.</span></span> <span data-ttu-id="4d694-123">詳細については、「[グループを比較する](../create-groups/compare-groups.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="4d694-123">To learn more, see [Compare Groups](../create-groups/compare-groups.md).</span></span>

- <span data-ttu-id="4d694-124">**暗号化:** 共有メールボックスから送信されたメールを暗号化することはできません。</span><span class="sxs-lookup"><span data-stu-id="4d694-124">**Encryption:** You can't encrypt email sent from a shared mailbox.</span></span> <span data-ttu-id="4d694-125">共有メールボックスには独自のセキュリティ コンテキスト (ユーザー名とパスワード) がないため、キーを割り当てることができないためです。</span><span class="sxs-lookup"><span data-stu-id="4d694-125">This is because a shared mailbox does not have its own security context (username/password) so it cannot be assigned a key.</span></span> <span data-ttu-id="4d694-126">2 人以上のユーザーがメンバーで、メンバーが独自のキーを使用して暗号化してメールを送受信した場合、他のメンバーはメールを読むことができる場合もあれば、できない場合もあります。メールがどのパブリック キーで暗号化されたかに依存します。</span><span class="sxs-lookup"><span data-stu-id="4d694-126">If more than one person is a member, and they send/receive emails they encrypted with their own keys, other members might be able to read the email and others might not, depending which public key the email was encrypted with.</span></span>

- <span data-ttu-id="4d694-127">**メールボックスの変換:** ユーザー メールボックスを、共有メールボックスに変換することができます。</span><span class="sxs-lookup"><span data-stu-id="4d694-127">**Mailbox conversion:** You can convert user mailboxes to shared mailboxes.</span></span> <span data-ttu-id="4d694-128">「[ユーザー メールボックスを共有メールボックスに変換する](convert-user-mailbox-to-shared-mailbox.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="4d694-128">See [Convert a user mailbox to a shared mailbox](convert-user-mailbox-to-shared-mailbox.md).</span></span>

- <span data-ttu-id="4d694-129">**管理者の役割:** グローバル管理者または Exchange 管理者の役割を持つユーザーは共有メールボックスを作成することができます。</span><span class="sxs-lookup"><span data-stu-id="4d694-129">**Admin roles:** Users with global admin or Exchange admin roles can create shared mailboxes.</span></span>

- <span data-ttu-id="4d694-133">**サインイン:** 共有メールボックスは、関連付けられたユーザー アカウントによる直接のサインインを目的としていません。</span><span class="sxs-lookup"><span data-stu-id="4d694-133">**Signing in:** A shared mailbox is not intended for direct sign-in by its associated user account.</span></span> <span data-ttu-id="4d694-134">共有メールボックス アカウントへのサインインは常にブロックし、ブロックしたままにしておく必要があります。</span><span class="sxs-lookup"><span data-stu-id="4d694-134">You should always block sign-in for the shared mailbox account and keep it blocked.</span></span>

- <span data-ttu-id="4d694-137">**メッセージの削除:** 残念ながら、ユーザーに対して共有メールボックス内にあるメッセージの削除を禁止することはできません。</span><span class="sxs-lookup"><span data-stu-id="4d694-137">**Message deletion:** Unfortunately, you can't prevent people from deleting messages in a shared mailbox.</span></span> <span data-ttu-id="4d694-138">これを回避する唯一の方法は、共有メールボックスの代わりに、Microsoft 365 グループを作成することです。</span><span class="sxs-lookup"><span data-stu-id="4d694-138">The only way around this is to create a Microsoft 365 group instead of a shared mailbox.</span></span> <span data-ttu-id="4d694-139">Outlook でのグループは、共有メールボックスと同様です。</span><span class="sxs-lookup"><span data-stu-id="4d694-139">A group in Outlook is like a shared mailbox.</span></span> <span data-ttu-id="4d694-140">この 2 つを比較するには、「[グループを比較する](../create-groups/compare-groups.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="4d694-140">For a comparison of the two, see [Compare groups](../create-groups/compare-groups.md).</span></span> <span data-ttu-id="4d694-141">グループの詳細については、「[グループの概要](https://support.microsoft.com/office/b565caa1-5c40-40ef-9915-60fdb2d97fa2)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="4d694-141">To learn more about groups, see [Learn more about groups](https://support.microsoft.com/office/b565caa1-5c40-40ef-9915-60fdb2d97fa2).</span></span>
