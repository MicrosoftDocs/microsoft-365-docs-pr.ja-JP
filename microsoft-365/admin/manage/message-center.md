# <a name="message-center"></a>メッセージ センター

新機能や変更された機能、計画メンテナンス、その他の重要なアナウンスなど、今後の変更を追跡するには、[メッセージ センター] に <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">移動します</a>。
  
メッセージ センターを開く方法:

::: moniker range="o365-worldwide"

- 管理センターで、[正常性メッセージ センター **]** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">に移動します</a>。

::: moniker-end

::: moniker range="o365-germany"

- 管理センター <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">で、[正常性</a>メッセージ センター **]** > **に移動します**。

::: moniker-end

::: moniker range="o365-21vianet"

- 管理センター <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">で、[正常性</a>メッセージ センター **]** > **に移動します**。

::: moniker-end

モバイル デバイスで [Microsoft 365 管理](https://go.microsoft.com/fwlink/p/?linkid=627216) アプリを使用してメッセージ センターを表示することもできます。これは、プッシュ通知を最新の状態に戻す最適な方法です。

メッセージ センターは、すべてのアクティブ なメッセージのビューを表形式で表示します。 既定では、リストの上部に最新のメッセージが表示されます。 [サービス] を **選択** すると、Microsoft 365 Apps、SharePoint Online などのさまざまなサービスのメッセージを表示できます。  [**タグ]** で、[管理者への影響 **]、[****データプライバシー** **]、[** 機能の更新 **]、[メジャー更新****]、[新機能]、[** 退職]、または [**ユーザー影響メッセージ] を選択** できます。 [**メッセージの状態]** で、[お気に **入り]、[****未読]、** または [**更新されたメッセージ] を選択** できます。

[アーカイブ] タブには、アーカイブしたメッセージが表示されます。 メッセージをアーカイブするには、メッセージ ウィンドウで [アーカイブ] を **選択します**。

### <a name="major-updates"></a>メジャー更新プログラム

主要な更新プログラムは、[タグ] ドロップダウンから [ **メジャー** 更新プログラム] **を選択して** 確認できます。

主要な更新プログラムは、アクションが必要な場合は少なくとも 30 日前に伝達され、次のものが含まれる場合があります。
  
- 受信トレイ、会議、委任、共有、アクセスなどの毎日の生産性の変更

- カスタマイズされた機能に影響を与える可能性があるテーマ、Web パーツ、その他のコンポーネントの変更

- 記憶域、ルール、アイテム、期間などの表示可能な容量の増減

- 製品のブランド化に対する次の変更点。

  - エンド ユーザーの混乱を引き起こす

  - ヘルプ デスク のプロセスと参照資料に変更が加わるか、または

  - URL の変更

- 新しいサービスやアプリケーション

- 管理者による対応が必要となる変更 (問題の予防または修正を除く)

- データの保存場所の変更
  
### <a name="preferences"></a>基本設定

管理が組織全体に分散されている場合は、すべての Microsoft 365 サービスに関する投稿を表示したくない場合や、表示する必要が生じない場合があります。 各管理者は、次の方法を実行できます。

- メッセージ センターに表示するメッセージを制御する基本設定を設定します。
- フィルター メッセージ
- メールの基本設定を設定して、すべてのメッセージの週次ダイジェスト、主要な更新プログラム専用のメール、およびデータ プライバシー メッセージのメールを受信します。  

::: moniker range="o365-worldwide"

1. メッセージ **センターの上部** にある [基本設定] を選択します。

2. [カスタム **ビュー] タブ** で、監視するサービスごとにチェック ボックスがオンに設定されている必要があります。 メッセージ センター ビューからフィルター処理するサービスのチェック ボックスをオフにします。

3. ダイジェスト メールは既定でオンになっており、プライマリの電子メール アドレスに届きます。 週ごとのダイジェストの受信を停止するには、[電子メール] タブの [メッセージ センターからメール通知を送信する] チェック ボックス **をオフにします**。 

   また、最大 2 つの電子メール アドレスをセミコロンで区切って入力することもできます。

   取得するメールと、選択したサービスの週ごとのダイジェストを選択することもできます。

4. [保存 **] を** 選択して変更を保持します。
  
::: moniker-end

::: moniker range="o365-germany"

1. メッセージ センターの上部で [ **メッセージ センターの設定の編集**] を選びます。

2. 監視する各サービスのトグルが **オン** に設定されていることを確認します。 トグルを使用して、メッセージ センタービューからフィルター処理するサービスの設定を [オフ] に変更します。

3. ダイジェスト メールは既定でオンになっており、プライマリの電子メール アドレスに届きます。 週ごとのダイジェストの受信を停止するには、[メッセージの週ごとのダイジェストを送信する] 設定を **[オフ** ] に **変更します**。 

   メジャー更新プログラムの電子メール通知は別に制御されます。 メジャー更新プログラムに関する電子メール通知を受け取る場合、[ **メジャー更新プログラムのメールを受信する**] を [ **オン**] にします。 主要な更新プログラムに関 **するメール** の受信を停止するには、設定を [オフ] に変更します。

   プライマリ メール アドレスを選択またはクリアできますが、変更は行えなくなっています。 毎週の電子メールの概要を送信する他の電子メール アドレスを指定するには、[自分のメッセージの週ごとのダイジェストを送信する] が **[オン** ] に設定されている必要 **があります**。 2 人を超えるユーザーがダイジェスト メールを取得する場合は、Microsoft 365 グループのメール アドレスまたは配布リストを入力します。
