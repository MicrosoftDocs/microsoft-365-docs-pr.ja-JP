# <a name="message-center"></a>メッセージ センター


新機能や変更された機能、計画されたメンテナンス、その他の重要なアナウンスなど、今後の変更を追跡するには、メッセージ センターに <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">移動します</a>。 
  
メッセージ センターを開く方法:

::: moniker range="o365-worldwide"

- 管理センターで、正常性メッセージ センター **に** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">移動します</a>。

::: moniker-end

::: moniker range="o365-germany"

- 管理センター <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">で、正常性</a>メッセージ **センターに** > **移動します**。

::: moniker-end

::: moniker range="o365-21vianet"
 
- 管理センター <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">で、正常性</a>メッセージ **センターに** > **移動します**。

::: moniker-end

モバイル デバイスで [Microsoft 365 Admin](https://go.microsoft.com/fwlink/p/?linkid=627216) アプリを使用してメッセージ センターを表示することもできます。これは、プッシュ通知を使用して最新の状態を続け取る最適な方法です。

### <a name="major-updates"></a>メジャー更新プログラム

主要な更新プログラムは、メッセージ センターの上部にある [重要度の高い] タブを選択して確認できます。

主要な更新プログラムは、アクションが必要な場合に少なくとも 30 日前に通信され、次の情報が含まれる場合があります。
  
- 受信トレイ、会議、委任、共有、アクセスなど、毎日の生産性に対する変更

- カスタマイズされた機能に影響を与える可能性があるテーマ、Web パーツ、その他のコンポーネントの変更

- 記憶域、ルールの数、アイテム、期間など、表示される容量の増減

- 製品ブランド化の変更。次の場合があります。

  - エンド ユーザーの混乱を引き起こす

  - ヘルプ デスクのプロセスと参照資料に変更を加え、

  - URL を変更する

- 新しいサービスやアプリケーション

- 管理者による対応が必要となる変更 (問題の予防または修正を除く)

- データの保存場所に対する変更
