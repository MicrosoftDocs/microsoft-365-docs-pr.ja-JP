# <a name="communication--people-experiences"></a>コミュニケーション–ユーザーエクスペリエンス

生産性スコアは、組織が Microsoft 365 とそれをサポートするテクノロジエクスペリエンスについての洞察による、デジタル変換への移行をサポートしています。 組織のスコアはユーザーと技術エクスペリエンスの測定を反映しており、同様の規模の別の組織によるベンチマークと比較できます。 コミュニケーションカテゴリは、ユーザーエクスペリエンス対策の一部です。 詳細については、「 [生産性スコアの概要」](productivity-score.md) と「 [Microsoft のプライバシー](https://privacy.microsoft.com/privacystatement)に関する声明」を参照してください。

## <a name="why-your-organizations-communication-score-matters"></a>組織の通信スコアが重要な理由 

Microsoft は、さまざまなコミュニケーションニーズがあることを認識しています。 質問への迅速な対応を得るには、インスタントメッセージを送信することをお勧めします。 経営陣に進捗状況の更新を送信する場合は、電子メールメッセージを選択することができます。 幅広い対象ユーザーにアクセスするには、コミュニティメッセージを投稿することを選択できます。 Microsoft 365 は、すべてのユーザーのニーズに合わせて通信モードでこの柔軟性を実現します。 調査によれば、リアルタイム通信ツールを使用することで、より統一された組織を作成し、場所に関係なく士気を構築していることがわかります。 [証拠を参照してください](https://go.microsoft.com/fwlink/?linkid=2127669)。

## <a name="how-we-calculate-the-communication-score"></a>コミュニケーションスコアの計算方法

コミュニケーションを目的として、組織内のコミュニケーションに関連する主要な指標と共に、組織のスコアを計算するためにこれらの指標を使用したスコアフレームワークと組み合わせた、主な洞察を提供します。

### <a name="primary-insight"></a>主要な洞察

Microsoft 365 では、複数のモードを提供することにより、すべてのユーザーのコミュニケーションスタイルを柔軟に調整することができます。 主要な洞察では、複数のモードを使用して通信する組織内のユーザーの測定を行います。

:::image type="content" source="../../media/primaryinsight.png" alt-text="コミュニケーションモードに対する主要な洞察。":::

1. **ヘッダー:** 複数の通信モード (つまり、次のもの) を使用しているユーザーの主要な指標を示します。

   - Exchange によるメールの送信
   - Teams によるメッセージの送信
   - Yammer でコミュニティに投稿する

   この主な指標は、2つ以上の Exchange、Teams、または Yammer を使用しているすべてのユーザーに対する割合として示されています。 

2. **本文:** 組織内のユーザーにとって、コミュニケーションのモードの柔軟性がどのように役立つかについて詳しく説明します。

3. **現在の状態の視覚化:**

   - 青い部分がヘッダーで表されるパーセンテージを表す水平バー
   - ヘッダーで表されるパーセンテージの計算に使用される (分子または分母) を強調表示します。
     - **分子**: 過去28日間に複数の通信モードを使用しているユーザーの数
     - **分母**: 過去28日以内に複数の通信製品に対して有効としてマークされたユーザーの数
   - 重要な指標の **ピアベンチマーク** 値もパーセンテージとして表示されます。

4. **コミュニケーションリソースの表示:** ここをクリックすると、通信に関連するビデオや記事の形式ですべてのサポートコンテンツが登録されます。 これにより、コミュニケーションのためのカスタムプレイリストに移動し、その後、すべての生産性スコアカテゴリに移動することができます。

#### <a name="trend-visualization-of-primary-insight"></a>主要な洞察の傾向を視覚化

 このグラフは、主な洞察 (つまり、過去180日間に複数の通信モードを使用した組織内のユーザー数) の主要指標の分子の傾向を示しています。 ここで、daily の値は、過去28日間に複数の形式の通信を使用しているユーザーの数を集計したものです。

:::image type="content" source="../../media/trend-communication.png" alt-text="2つ以上のモードを使用しているユーザーの数を示すグラフ。":::

### <a name="scoring-model"></a>スコアリングモデル

組織のコミュニケーションスコアは、1人の電子メール、チャット、コミュニティの投稿の間で、28日の時間帯に複数のモードを使用して、一貫したコミュニケーションが行われているかどうかを集約 (組織) レベルで測定します。  

スコアは個別のユーザーレベルでは提供されません。

## <a name="explore-how-your-organization-communicates"></a>組織がどのように通信するかを調べる

また、組織がどのように通信するかを把握するのに役立つ情報も提供します。 これらの追加指標は、生産性のスコアに直接影響を与えるものではありませんが、デジタル変換の一部としてのアクションプランの作成に役立てることができます。

### <a name="breakdown-of-communication-by-modes"></a>モード別の通信の内訳

:::image type="content" source="../../media/howpeoplecommunicate.png" alt-text="![通信モードの内訳]":::

1. **ヘッダー:** コミュニケーションで考慮されているさまざまな通信モードについての詳細な分類を提供することで、洞察の目的を強調しています。

2. **本文:** さまざまなモードを使用して知識を共有するための値についての情報を提供します。

3. **視覚エフェクト:** ブレークダウンは、各モードの使用を表します。 各バーの色付き部分と分数は、そのモードで有効になっているユーザー数の割合として、電子メール、メッセージ、またはコミュニティの投稿を送信したユーザーの数を表します。

   - **電子メールを送信するユーザー:** 色の部分と分数は、電子メールを送信している Exchange が有効になっているユーザーの割合を表します。 この分数は次のものから構成されています。
   
     - **分子**: 過去28日以内に電子メールを送信した人。
     - **分母**: 過去28日以内に Exchange が有効になっているユーザー。
     
   - **Microsoft Teams でメッセージを送信するユーザー:** 色の部分と分数は、メッセージを送信している Microsoft Teams が有効になっているユーザーのパーセンテージを表します。 この分数は次のものから構成されています。
   
     - **分子**: 過去28日以内に Microsoft Teams でメッセージを送信する人。
     - **分母**: 過去28日以内に Microsoft Teams で有効になっているユーザー。
     
   - **コミュニティに投稿する人:** 色の部分と分数は、コミュニティに投稿される Yammer が有効になっているユーザーの割合を表します。 この分数は次のものから構成されています。
   
     - **分子:** 過去28日以内に Yammer コミュニティに投稿された人。
     - **分母:** 過去28日以内に Yammer に対して有効になっているユーザー。
          
4. **関連するコンテンツを表示する:** このリンクを選択すると、丁合いされたビデオやその他の関連するヘルプコンテンツが表示されます。

### <a name="mentions-in-emails"></a>電子メール内の @mentions

:::image type="content" source="../../media/emailmentions.png" alt-text="![メンション付きメール]":::

1. **ヘッダー:** 過去28日以内に開始された新しい電子メールスレッドの @mentions が含まれている場合の、応答率の増加を強調表示します。

2. **本文:** 電子メールで @mentions を使用する場合の値について説明します。 さらに、@mentions を使用するユーザーは、過去28日以内に電子メールを送信したすべてのユーザーに対する割合として表現されます。

3. **視覚エフェクト:** 新しい電子メールスレッドの応答速度を、@mentions たかどうかに基づいて分割します。

   - **\@ メンション付きの電子メールへの応答:** 色付きの部分と分数は、@mentions を含む新しい電子メールスレッドの応答速度を表します。 この分数は次のものから構成されています。
   
     - **分子:** 過去28日以内に開始され、応答を受信した @mentions を含む新しい電子メールスレッド。
     - **分母:** 過去28日以内に開始された @mentions を含む新しい電子メールスレッド。
     
   - **\@ メンションのない電子メールへの応答:** 色付きの部分と分数は、@mentions を含む新しい電子メールスレッドの応答速度を表します。 この分数は次のものから構成されています。
   
     - **分子:** 開始され、過去28日以内に応答を受信した @mentions を含まない新しい電子メールスレッド。
     - **分母:** 過去28日以内に開始された @mentions を持たない新しい電子メールスレッド。
     
4. **関連するコンテンツを表示する:** このリンクを選択すると、丁合いされたビデオやその他の関連するヘルプコンテンツが表示されます。
