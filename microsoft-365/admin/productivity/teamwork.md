# <a name="teamwork--people-experiences"></a>チームワーク - 人のエクスペリエンス

生産性スコアは、Microsoft 365 の使用とそれをサポートするテクノロジ エクスペリエンスを通じて、組織のデジタル変換への取り組みについての洞察を提供します。 組織のスコアは、人とテクノロジのエクスペリエンスの測定値を反映し、自分と同様の組織からのベンチマークと比較できます。 チームワーク カテゴリは、ユーザー エクスペリエンスに該当する測定値の一部です。 詳細については、生産性スコアの概要 [を](productivity-score.md) 確認し、Microsoft のプライバシーに関する声明 [を参照してください](https://privacy.microsoft.com/privacystatement)。

## <a name="prerequisites"></a>前提条件

チームワークの洞察を開始するには、組織内のユーザーに次のライセンスが必要です。

- Microsoft Teams
- SharePoint
- Exchange Online

詳細については、「ライセンスをユーザーに [割り当てる」を参照してください](../manage/assign-licenses-to-users.md)。

過去 28 日間に少なくとも 1 回、上記の製品でユーザーがアクティブにされた後、分析情報の表示が開始されます。

## <a name="why-your-orgs-teamwork-score-matters"></a>組織のチームワーク スコアが重要な理由

組織の生産性の基本的な柱は、共通の目標を持つ人々のグループが互いに作業し、集合的な成功のための共通のリソースを持つ場合です。 調査では、ユーザーが情報を共有し、共有ワークスペースで共同作業を行う場合、週に最大 4 時間の節約が行われます。 関連するドキュメントの検索、以前のディスカッションのコンテキストの検索、共有目標の実現を行います。 証拠を確認[する](https://www.microsoft.com/microsoft-365/blog/wp-content/uploads/sites/2/2019/04/Total-Economic-Impact-Microsoft-Teams.pdf)

## <a name="how-we-calculate-the-teamwork-score"></a>チームワーク スコアの計算方法

組織内のこのカテゴリの主要な指標を含むエクスペリエンスに関する主要な分析情報を提供します。 次に、以下に示すスコアリング フレームワークを使用して、組織のスコアを計算します。

### <a name="primary-insight"></a>主な分析情報

主要な分析情報は、Microsoft Teams で電子メールとメッセージを使用して通信し、共有ワークスペース内のクラウド上のコンテンツを操作しているすべてのユーザーを確認します。 Microsoft 365 では、Microsoft 365 グループは、ユーザーがグループ メールボックスにメールを送信し、SharePoint チーム サイトでファイルを共有し、Microsoft Teams を介してチャネル メッセージを送信する機能を備え、共有ワークスペースに集めることもできます。

:::image type="content" source="../../media/teamwork-score.png" alt-text="チームワーク スコアに関する主要な分析情報を示す図。":::

1. **ヘッダー:** 次のいずれかのアクティビティを実行している組織のユーザーの主要な指標を提供します。
      - Exchange を介してグループ メールボックスに電子メールを送信する。
      - Teams を介してチャネル メッセージを送信する
      - SharePoint チーム サイトでのコンテンツの読み取りおよび作成 (総称してコンテンツ操作と言います)。

        (共有ワークスペースの中または外部で) 次のアクティビティを実行している、組織のすべてのユーザーの割合:
        - Exchange 経由で電子メールを送信する。
        - Microsoft Teams でのメッセージの送信 (チャットまたはチャネル メッセージ)。
        - OneDrive または SharePoint でのコンテンツの読み取りおよび作成。

            また、Exchange、Microsoft Teams、または SharePoint のいずれかのサービスにアクセスできます。

1. **本文:** 共有ワークスペース内でコンテンツの通信や操作を行う場合に、組織の生産性に良い結果を与える方法について詳しい情報を提供します。
2. **視覚化 (現在の状態):**
      - ヘッダーで表されるパーセンテージを青色の部分が表す水平バー
      - ヘッダーに表示されるパーセンテージの計算に使用される分数 (numerator/分母) を強調表示します。
        - Numerator: 組織内で Exchange を介してグループ メールボックスに電子メールを送信するユーザー、または Teams を介してチャネル メッセージを送信しているユーザー、または SharePoint チーム サイトのコンテンツの読み取りおよび作成を行うユーザーの数。
        - 分母: Exchange を介して電子メールを送信したり、Microsoft Teams でメッセージ (チャットまたはチャネル メッセージ) を送信したり、OneDrive または SharePoint でコンテンツを読んで作成したり、Exchange、Microsoft Teams、または SharePoint のいずれかのサービスにアクセスできる、組織のユーザーの数。
   - キー メトリックのピア ベンチマーク値もパーセンテージで表示されます。
3. **チームワークに関するリソースを表示します。** ヘルプ コンテンツを表示するには、このリンクを選択します。

### <a name="scoring-framework"></a>スコアリング フレームワーク

組織のチームワーク スコアは、ユーザーが過去 28 日以内に共有ワークスペースでファイル アクティビティを常に通信または関与するかどうかを集計 (組織) レベルで測定します。

スコアは個々のユーザー レベルでは提供されない。

## <a name="explore-more-about-teamwork-in-your-organization"></a>組織のチームワークの詳細を確認する

また、組織内のユーザーがどのように一緒に作業する方法に関する追加情報も提供します。 これらの追加の指標は生産性スコアに直接貢献するのではなく、デジタル変換の一環としてアクション プランを作成する場合に関連します。

### <a name="breakdown-of-how-people-engage-in-shared-workspaces"></a>ユーザーが共有ワークスペースに参加する方法の詳細

:::image type="content" source="../../media/people-sharedworkspace-engagement.png" alt-text="組織のユーザーが共有ワークスペースに参加する方法を示す図。":::

1. **ヘッダー:** 測定されるさまざまな種類のチームワークの詳細な内訳を示します。
2. **本文:** チームの効率を高め、共有ワークスペースでの作業の価値に関する情報を提供します。
3. **視覚化:** 視覚エフェクトは、コンテンツの通信または操作を行っているユーザーが共有ワークスペースでどの程度の作業を行っているのかを示しています。次に示します。
      - **電子メールの送信**: 色付き部分と分数は、グループ メールボックスにメールを送信するユーザーの割合を表します。 分数は、次の構成で構成されます。
        - Numerator: 過去 28 日以内にグループ メールボックスにメールを送信するユーザー。
        - 分母: 過去 28 日以内にメールを送信するユーザー。 これは、コミュニケーション生産性スコアの主要な分析情報でメールの送信としてマークされているユーザーのグループと同じです。
      - **メッセージの送信**: 色付き部分と分数は、Microsoft Teams のチャネルでメッセージを送信するユーザーの割合を表します。 分数は、次の構成で構成されます。
        - Numerator: 過去 28 日以内にチャネル メッセージを送信するユーザー。
        - 分母: 過去 28 日以内にチャットまたはチャネル メッセージを送信するユーザー。 これは、生産性スコアの通信カテゴリの主要な分析情報で Microsoft Teams でメッセージの送信としてマークされているユーザーのグループと同じです。
    - **コンテンツの** 作成 : 色付き部分と分数は、Microsoft 365 SharePoint チーム サイトでコンテンツを読み書きするユーザーの割合を表します。
        - Numerator: Microsoft 365 グループに接続されたチーム サイトでコンテンツを読み書きまたは作成するユーザーの数。
        - 分母: 過去 28 日間に OneDrive または SharePoint サイトで任意の種類のコンテンツを読み取りまたは作成した SharePoint にアクセスできるユーザーの数。
4. **関連するコンテンツを表示します。** ヘルプ コンテンツを表示するには、このリンクを選択します。

### <a name="breakdown-of-workspace-engagement-by-size-and-age"></a>サイズと年齢別のワークスペースエンゲージメントの内訳

:::image type="content" source="../../media/workspace-engagement-bysizeage.png" alt-text="ワークスペースでのエンゲージメントをサイズと年齢別に分類して示すグラフ。":::

1. **ヘッダー:** ワークスペースでのエンゲージメントの分類を示します。ワークスペース内のメンバー数のサイズ、およびワークスペースの保存時間 (月) によって分類されます。
2. **本文:** 組織内のユーザーに対して、より効果的なチームワークを促進するために必要なワークスペースのみを維持する必要があるという価値についての情報を提供します。
3. **視覚化:** 契約の内訳は、2 次元にわたる温度マップの形で表示されます。

      - **ワークスペースのサイズ:** ワークスペースは、メンバー数に基づいて 2 ~ 10 人、11 ~ 100 人、100 人を超える 100 人という 3 つのカテゴリに分類されます。 All &quot; カテゴリ &quot; には、すべてのサイズ カテゴリが含まれます。
      - **ワークスペースの保存時間:** ワークスペースは、ワークスペースが最初に作成された月数によって分類されます。 All &quot; カテゴリ &quot; には、すべての年齢カテゴリが含まれます。

        グラフの各セルには、そのカテゴリに属するエンゲージされたワークスペースの割合に基づいて数値と色が表示されます。 ワークスペースのカテゴリは、そのセルの交差部分に表示される年齢とサイズに基づいて作成されます。 たとえば、11 ~ 100 人と 4 ~ 12 か月の交点にあるセルの値が 52% の場合、4 ~ 12 か月の間にメンバーが 11 ~ 100 人のワークスペースの 52% が何らかの形で関与しているという意味です。 パーセンテージは次のように計算されます。

        - **Numerator:** 過去 28 日間のコミュニケーション (電子メールおよびチャネル メッセージ) またはコンテンツのやり取りという形で関与するワークスペース
        - **分母:** 過去 28 日間に組織で利用可能なすべてのワークスペース

1. **関連するコンテンツを表示します。** ヘルプ コンテンツを表示するには、このリンクを選択します。

### <a name="breakdown-of-workspaces-by-level-of-engagement"></a>エンゲージメント のレベル別のワークスペースの内訳

:::image type="content" source="../../media/workspace-by-engagement.png" alt-text="エンゲージメント別のワークスペースのグループ別の内訳を示す図。":::

1. **ヘッダー:** グループ の電子メール、チャネル メッセージ、コンテンツの操作を使用して、エンゲージメントのレベル別に分いたワークスペースの詳細を提供します。
2. **本文:** 共有ワークスペースでの一貫性のあるエンゲージメントの価値に関する情報を提供し、チームワークの効率を高めるのに役立ちます。
3. **視覚化:** 1 週間あたりのエンゲージメントの強さに基づいて、組織内のワークスペースのビューを提供します。 ビューには、チームワーク内で測定されたさまざまなアクティビティの種類に対する配布と、エンゲージメントが含まれます。これには、次のカテゴリが含まれます。
      - **グループ メール:** 過去 28 日間に 1 週間に 1 日/2 ~ 3 日/4 日以上のグループ メール アクティビティがないワークスペースの割合。
      - **チャネル メッセージ:** 過去 28 日間にチャネル メッセージが 1 週間に 1 日/1 日/2 ~ 3 日/4 日以上含めないワークスペースの割合。
      - **コンテンツの読み取りまたは作成:** 過去 28 日間にコンテンツの読み取りまたは作成が 1 週間に 1 日/2 ~ 3 日/4 日間を超える日数を持つワークスペースの割合。
4. **関連するコンテンツを表示します。** ヘルプ コンテンツを表示するには、このリンクを選択します。
