# <a name="protect-yourself-against-phishing-and-other-attacks"></a>フィッシングなどの攻撃から自分自身を保護する

Microsoft 365 には攻撃から保護する多くの保護が含まれていますが、オンライン攻撃のリスクを制限して最小限に抑えるために自分で行える操作があります。

- **スパムまたは迷惑メール** 迷惑メールを受信する理由は多く、すべての迷惑メールが同じではない場合があります。 ただし、迷惑メールをフィルター処理することで、ユーザーに提供される情報を減らし、攻撃のリスクを軽減できます。
- **フィッシング** フィッシング詐欺は、正当に見えるが、個人情報を取得したり、お金を盗もうとしたりしようとするメールです。
- **スプーフィング** 詐欺詐欺者は、スプーフィングと呼ばれる手法を使用して、自分からメールを受信した場合と同様に表示することもできます。 
- **マルウェア** とは、コンピューターにインストールできる悪意のあるソフトウェアです。通常は、リンクをクリックするか、電子メールからドキュメントを開いた後にインストールされます。 マルウェアにはさまざまな種類がありますが (たとえば、コンピューターが引き継がれやランサムウェアなど)、マルウェアを使用する必要はありません。 

## <a name="best-practices"></a>ベスト プラクティス

### <a name="reduce-spam-mail"></a>スパム メールを減らす

スパムを減 [らす方法については、次の 10 のヒントに従ってください](https://support.microsoft.com/en-us/office/10-tips-on-how-to-help-reduce-spam-55f756e8-688b-41c3-a086-8f68ccc592f6)。

### <a name="report-it"></a>報告してください。

受信したフィッシングメールまたは他の詐欺メールを報告します。 メッセージを選択し、リボンの **[メッセージの報告** ] を選択します。
迷惑メールとフィッシングメールの [報告に関する詳細を次に示します](https://support.office.com/article/Use-the-Report-Message-add-in-b5caa9f1-cdf3-4443-af8c-ff724ea719d2)。

### <a name="avoid-phishing"></a>フィッシングを回避する

- 個人またはアカウント情報の送信を求めるメールには返信しない。
- 不審に思われるメールを受け取った場合や、この種類の情報を求めるメールを受け取った場合は、会社の Web サイトにアクセスできる可能性があるリンクをクリックする必要があります。
- 疑わしいメールに添付されたファイルを開かれません。
- メールが会社から送信されたと思う場合は、電話または Web ブラウザーで会社のカスタマー サービスに連絡して、メールが正当なものか確認してください。
- Web でメールの件名行の後に「デマ」という単語を検索し、他の誰かがこの詐欺を報告したのかを確認します。

不正使用、フィッシング、スプーフィングに関する 5 種類の一般的な詐欺 [について説明します](https://support.office.com/article/Deal-with-abuse-phishing-or-spoofing-in-Outlook-com-0d882ea5-eedc-4bed-aebc-079ffa1105a3)。

### <a name="make-sure-your-emails-look-legitimate-to-others"></a>メールが他のユーザーに対して正当に表示される

デジタル署名を追加して、顧客が自分のコミュニケーションを信頼し、それがユーザーから提供されたことを証明します。 「デジタル [署名を使用してメッセージをセキュリティで保護する」を参照してください](https://support.office.com/article/secure-messages-by-using-a-digital-signature-549ca2f1-a68f-4366-85fa-b3f4b5856fc6)。
